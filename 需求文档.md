# 我的家 - 家庭网站开发文档

## 项目概述
一个温馨的家庭展示网站，用于展示家庭成员和家庭趣事，风格参考苹果官网的清新简约设计，并加入魔幻星星特效。

---

## 一、网站定位

### 目标受众
- 主要：家庭成员
- 次要：作为网页开发学习案例展示

### 网站性质
- 展示型网站
- 无需隐私保护
- 响应式设计（手机/平板/电脑）

---

## 二、功能需求

### 2.1 家庭成员展示（家谱树）
- **展示形式**：树状结构，按辈分分支组织
- **成员信息**：
  - 姓名
  - 照片
  - 简介
  - 工作
  - 爱好
- **家族结构**：
  - 第一代：爷爷
  - 第二代：爸爸、伯伯、叔叔、姑姑
  - 第三代：用户本人（初中生）、妹妹（小学生）
- **交互**：点击成员可查看详情

### 2.2 家庭趣事
- **内容形式**：图文混合
- **分类方式**：按时间排序
- **后续维护**：支持添加新内容

---

## 三、视觉设计

### 3.1 整体风格
- 温馨亲切
- 苹果官网风格：清新、简约、突出重点
- 大量留白，视觉舒适

### 3.2 特效
- **鼠标跟随星星特效**：
  - 鼠标移动时产生五颜六色的星星
  - 星星往下坠落
  - 魔幻梦幻感

### 3.3 色彩
- 无特定要求，由设计师搭配
- 建议使用暖色调体现温馨感

---

## 四、页面结构

```
🏠 我的家（首页）
├── 🌳 家谱树（成员展示）
│   └── 点击成员 → 查看详情弹窗
├── 📖 家庭趣事（时间线展示）
└── 📞 联系我们（可选）
```

---

## 五、技术方案

| 技术 | 选择 | 说明 |
|------|------|------|
| 前端框架 | Vue 3 | 轻量、易上手，数据管理方便 |
| 样式方案 | Tailwind CSS | 快速构建响应式布局 |
| 特效实现 | 原生 JavaScript + CSS动画 | 星星特效 |
| 家谱树 | CSS Flexbox + SVG连线 | 清晰展示家族关系 |
| 构建工具 | Vite | 开发速度快 |

---

## 六、开发阶段

### Phase 1: Demo版本
- 基础页面结构
- 家谱树展示（示例数据）
- 家庭趣事时间线（示例数据）
- 鼠标星星特效
- 响应式布局

### Phase 2: 后续优化
- 添加真实家庭成员数据和照片
- 完善交互效果
- 根据反馈调整设计
